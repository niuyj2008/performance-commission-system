## 下一步操作

### 1. 补充项目基本信息
登录系统后，为每个项目补充：
- 建筑面积
- 建筑类型
- 是否有地下室
- 是否有人防
- 是否绿建项目
- 其他必要信息

### 2. 配置空调面积表
对于需要计算空调提成的项目：
1. 进入项目详情页面
2. 点击"配置空调面积表"
3. 添加空调类型和面积
4. 保存后系统自动计算部门分配

### 3. 配置发放节点
为项目设置提成发放计划：
1. 在第一级点击"配置发放节点"
2. 添加发放阶段（如：施工图 75%）
3. 设置发放日期和比例
4. 保存后自动刷新第二级

### 4. 分配参与人员
在第三级进行个人分配：
1. 点击"分配参与人员"
2. 选择参与员工（可跨部门）
3. 输入分配金额
4. 系统自动验证当期限额
5. 保存并关联发放节点
