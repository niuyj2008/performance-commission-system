========================================
  绩效提成管理系统 - 快速开始
========================================

欢迎使用绩效提成管理系统！

本系统采用最简单的Docker部署方式，
只需3步即可完成部署。


第一步：安装Docker
========================================

Windows用户：
  1. 访问 https://www.docker.com/products/docker-desktop
  2. 下载 Docker Desktop for Windows
  3. 双击安装，一路"下一步"
  4. 安装完成后重启电脑

Mac用户：
  1. 访问 https://www.docker.com/products/docker-desktop
  2. 下载 Docker Desktop for Mac
  3. 拖动到应用程序文件夹
  4. 启动 Docker Desktop


第二步：启动系统
========================================

Windows用户：
  双击"启动系统.bat"文件

Mac/Linux用户：
  双击"启动系统.sh"文件
  或在终端运行: ./启动系统.sh

等待1-2分钟，系统会自动启动并打开浏览器。


第三步：登录系统
========================================

浏览器会自动打开系统页面：
  http://localhost:3000

如果没有自动打开，请手动访问上述地址。

默认管理员账号：
  用户名: admin
  密码: admin123

⚠️ 重要：首次登录后请立即修改密码！


常用操作
========================================

启动系统：
  Windows: 双击"启动系统.bat"
  Mac/Linux: 运行 ./启动系统.sh

停止系统：
  Windows: 双击"停止系统.bat"
  Mac/Linux: 运行 ./停止系统.sh

查看日志：
  Windows: 双击"查看日志.bat"

备份数据：
  Windows: 双击"备份数据.bat"
  Mac/Linux: 复制 data 文件夹


遇到问题？
========================================

1. 确保Docker Desktop已启动
2. 查看"部署指南.md"中的"常见问题"章节
3. 运行"查看日志.bat"查看错误信息
4. 联系技术支持

系统要求
========================================

- CPU: 2核或以上
- 内存: 2GB或以上
- 硬盘: 20GB可用空间
- 操作系统: Windows 10/11, macOS 10.15+, Ubuntu 20.04+
