# 绩效提成管理系统 - 部署检查清单

## 📋 部署前检查

### 硬件要求
- [ ] CPU: 2核或以上
- [ ] 内存: 2GB或以上  
- [ ] 硬盘: 20GB可用空间
- [ ] 网络: 稳定的局域网连接

### 软件要求
- [ ] 操作系统: Windows 10/11, macOS 10.15+, 或 Linux
- [ ] Docker Desktop已安装（本地部署）
- [ ] 浏览器: Chrome, Firefox, Safari, Edge（最新版本）

### 文件准备
- [ ] 所有系统文件已复制到目标位置
- [ ] `Dockerfile` 文件存在
- [ ] `docker-compose.yml` 文件存在
- [ ] `server/` 文件夹完整
- [ ] `public/` 文件夹完整
- [ ] `data/` 文件夹存在
- [ ] 启动脚本文件存在
