## 数据来源

**Excel文件**: `ALL奖金发放表2025年7月8日含预支（20250716).xlsx`

**数据提取方式**:
- 项目名称: 从Excel第2列提取
- 本期实发金额: 从Excel第9列提取
- 发放比例: 从Excel第6-8列提取 (上期/本期/总比例)
- 部门分配: 从Excel第15-24列提取 (各部门金额)

## 导入脚本

**文件**: `import-project-commissions-2025h1.js`

**功能**:
1. 读取项目编码,查找项目ID
2. 创建或查找发放节点
3. 插入或更新部门分配金额

**部门ID映射**:
```javascript
{
  'arch': 5,      // 建筑部
  'structure': 6, // 结构部
  'water': 7,     // 给排水部
  'electric': 8,  // 电气部
  'hvac': 9       // 空调部
}
```

## 数据验证

### 验证方法
```bash
# 查看项目的部门分配
sqlite3 data/database.sqlite "
SELECT 
  p.code,
  p.name,
  d.name as department,
  dc.amount
FROM projects p
JOIN department_commissions dc ON p.id = dc.project_id
JOIN departments d ON dc.department_id = d.id
WHERE p.code LIKE '2025H1%'
ORDER BY p.code, d.id;
"

# 查看发放节点
sqlite3 data/database.sqlite "
SELECT 
  p.code,
  ps.stage_date,
  ps.stage_name,
  ps.previous_ratio,
  ps.current_ratio,
  ps.total_ratio
FROM projects p
JOIN payment_stages ps ON p.id = ps.project_id
WHERE p.code LIKE '2025H1%'
ORDER BY p.code;
"
```
